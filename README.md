# US-Loans-Project

This Power BI project analyzes various aspects of bank loans to provide visual and connections between different kind of Loans.
This Project is made of three Parts.

Dashboard 1: Summary
  - General information about Loans, their total amount, total funded amount, recieved amount and an average interest rate.
  - Gauge Charts of Good and Bad loans and their percantage as well as funded adn received amount
  - Table focused on showing loans by their status

Dashboard 2: Overview
  - Monthly trends by issue dates
  - Loans purpose brakedown with a Line Chart
  - Home Ownership Analysis using Tree Map that showcases if Landers are Homeowners, have a Mortgage or live with Rent
  - Able to filter data using Parameter fields provided on left side of report where you can choose different Loan Types as well as filter by Total amount received/Funded

Dashboard 3: Details
  - Detailed information on every Loan
  - Able to filter data by Loan Types and Loan Grades using Slicers on left side of the Dashboard
